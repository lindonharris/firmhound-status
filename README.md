# Firmhound Status

This repository contains the status page for Firmhound services, powered by [Upptime](https://upptime.js.org).

## Monitored Services

| Service | URL |
|---------|-----|
| API | https://api.firmhound.com/health |
| Developer Portal | https://developers.firmhound.com |
| Marketing Site | https://firmhound.com |

## Status Page

Visit [status.firmhound.com](https://status.firmhound.com) for real-time status.

## How It Works

- GitHub Actions checks all endpoints every 5 minutes
- Uptime data is stored in this repository
- Status page is hosted on GitHub Pages
- Incidents are managed via GitHub Issues
